# 安装与更新

## 下载

从 Forsion 官网下载页获取对应平台的安装包:

| 平台 | 安装包 | 说明 |
|------|--------|------|
| macOS | `.dmg` | 拖进"应用程序"即可 |
| Windows | 安装程序 | 按向导安装 |
| Linux | `.AppImage` | 赋予执行权限后直接运行,无需安装 |

## 首次启动

第一次打开 Forsion 会进入引导流程:

1. **选择工作区目录** — 你的笔记与文件的存放位置(默认在用户目录下)。这是一个普通文件夹,随时可以用 Finder / 资源管理器直接查看。
2. **连接模型** — 登录 Forsion 账号直接使用云端模型,或填入自己的 API Key。也可以先跳过,进应用后在设置里配置,详见[模型与接入方式](../chat/models-and-providers.md)。
3. **挑个主题** — 后续随时可换,见[主题与外观](../customization/themes.md)。

## 自动更新

- **Windows / Linux**:应用内检测到新版本后自动下载,重启即完成升级。
- **macOS**:应用内提示新版本,点击后前往下载安装(受系统签名机制限制,不做静默替换)。

版本更新内容可在 设置 → 关于 里查看更新日志。

## 中国大陆网络优化

如果你在中国大陆网络环境下遇到下载依赖、安装组件缓慢的问题,可以在设置中开启**镜像源**,应用会自动改用国内镜像获取运行环境与依赖。

> 💡 **提示**:首次使用某些功能(如本地运行环境)时,应用会自动检测缺失的组件并引导你安装,按提示操作即可。

## 命令行工具

桌面端可以一键把 `tangu` 命令安装到终端,并随应用更新自动保持最新。详见[命令行(Tangu CLI)](../reference/cli.md)。
