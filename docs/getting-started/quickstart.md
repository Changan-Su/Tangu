# 快速上手

装好了?十分钟把核心流程跑一遍。

## 1. 连上模型

打开 设置 → AI → 模型。你有几种选择,任选其一:

- **登录 Forsion 账号** — 最省事,直接用官方云端模型,不用管 Key。
- **自带 API Key** — 支持主流模型服务商,填入 Key 即用。
- **订阅登录** — 已有 Claude / ChatGPT 订阅?直接 OAuth 登录,用订阅额度跑 Forsion。

细节见[模型与接入方式](../chat/models-and-providers.md)。

## 2. 第一次对话

在输入框里随便聊点什么。你会看到:

- 回复**流式**出现,长思考的模型会先折叠展示"思考过程";
- AI 需要动手时(跑命令、读写文件)会弹出**审批卡**,你批准了才执行;
- 消息悬浮后有 复制 / 重新生成 / 分支 / 编辑 等操作。

> 💡 **提示**:让 AI 干复杂活之前,可以打开**计划模式**——它会先给出完整计划,你点头后才动手。见[工具与审批](../chat/tools-and-approvals.md)。

## 3. 记第一篇笔记

切到 **Amadeus** 空间,新建一篇笔记:

- 直接写 Markdown,`#` + 空格变标题,`[[` 呼出双链;
- 输入 `/` 打开斜杠菜单,可以插入多维表、白板、代码块等;
- 一切内容都是你工作区目录里的 `.md` 文件。

见 [Amadeus 总览](../amadeus/overview.md)。

## 4. 让 AI 主动干活

试试自动化:打开**自动化**空间,建一条规则,比如"每天早上 9 点让助手汇总今天的日程发给我"。结果会出现在**收件箱**。见[自动化](../spaces/automation.md)。

## 5. 逛一圈市场

主界面打开**应用市场**:技能、代理、插件、空间、主题,全部一键安装。见[应用市场](../customization/market.md)。
