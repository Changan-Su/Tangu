# 群聊模式

一个人的智慧有边界,一桌 AI 的讨论没有。群聊模式让多个 Agent 就同一话题**轮流发言、互相回应**,适合头脑风暴、方案评审、多视角分析。

## 开始一场群聊

新建会话时选择群聊,挑选参与的 Agent(每位都带着自己的人格与记忆上场),抛出话题即可。

## 讨论是怎么进行的

- Agent 们**按轮次发言**,每条消息都清楚标注发言者(名字与配色);
- 后发言者能看到前面所有人说了什么,会赞同、反驳、补充;
- 每轮结束后参与者**投票**决定继续讨论还是收场,聊天区会显示票型;
- 讨论结束后你可以继续追问任何一位。

## 你的角色

你不是旁观者:

- 随时插话,把讨论拉回正题或抛出新信息;
- 用 **@某个 Agent** 点名要求它回应;
- 手动停止讨论。

> ⚠️ **注意**:群聊在本地模式下可用。

> 💡 **提示**:参与者的多样性决定讨论质量——给 Agent 们写差异明显的[人格设定](../agents/overview.md)(一个激进、一个保守、一个只关心成本),比三个同质助手有价值得多。
