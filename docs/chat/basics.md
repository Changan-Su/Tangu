# 对话基础

## 消息是怎么呈现的

- **流式输出** — 回复逐字出现,随时可点停止。
- **思考过程** — 推理型模型的思考默认折叠为一行(✦ 思考过程),点开可看全文。
- **工具调用** — AI 动手干活(读文件、跑命令等)时,每一步以工具卡片内联展示,连续的步骤自动折叠成组,点开看详情。见[工具与审批](tools-and-approvals.md)。
- **待办与计划** — 复杂任务中 AI 会展示自己的待办清单和计划卡片,进度一目了然。

## 消息操作

悬浮在消息上可以:

| 操作 | 说明 |
|------|------|
| 复制 | 复制整条内容 |
| 重新生成 | 对最后一条回复不满意?再来一次 |
| 分支 | 从这条消息岔出一个新会话,原会话不受影响——适合"换个方向试试" |
| 编辑 | 修改你自己发过的消息并重跑 |
| 朗读 | 配置了语音(TTS)后出现,把回复读给你听 |

## 运行中插话

AI 干活的过程中你可以**继续发消息**:补充要求、纠正方向,不用等它跑完。新指示会在合适的节点被纳入。

## 网络波动与自动重试

请求因网络抖动失败时,引擎会**自动重试**(最多 3 次,间隔 1.5 / 3 / 4.5 秒),聊天区会出现琥珀色提示,例如:

> ⟳ 网络波动,3.0 秒后自动重试(第 2/3 次)

重试成功后提示自动消失。彻底失败时错误会以易读的形式给出(如"无法连接模型服务""连接中途断开"),括号里保留原始错误便于排查。常见错误的含义见[常见问题](../reference/faq.md)。

> ⚠️ **注意**:如果回复已经输出到一半连接才断开,为避免内容重复,这次不会自动重试——点"重新生成"即可。

## 快捷键

常用操作均有快捷键,可在 设置 → 选项 → 快捷键 中查看和自定义。
