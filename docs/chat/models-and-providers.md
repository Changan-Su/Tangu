# 模型与接入方式

Forsion 不绑定任何一家模型。四种接入方式可以并存,配置入口都在 设置 → AI → 模型。

## 方式一:Forsion 账号(最省事)

登录 Forsion 账号即可直接使用官方云端模型,无需申请和管理 API Key。默认的模型选择由官方持续调优(对话、后台任务、生图各有默认档),保持默认即可自动跟随推荐;想固定用某一款也可以手动指定。

## 方式二:自带 API Key

支持主流模型服务商——填入对应的 API Key 即可使用该家的全部可用模型。多个服务商可以同时配置,不同会话、不同 Agent 可以各用各的。

## 方式三:订阅登录

已经付费订阅了 Claude 或 ChatGPT?可以直接用订阅账号 OAuth 登录,用**订阅额度**(而非按量 API 计费)驱动 Forsion。

> ⚠️ **注意**:订阅登录仅在本地模式下可用。

## 方式四:自定义服务商

任何兼容 OpenAI 接口规范的服务都能接入:填入地址与 Key,即可把私有部署、中转服务、或本机跑的模型(如 Ollama)当作一个服务商使用。

## 按 Agent 指定模型

每个 Agent 可以单独设定自己的模型——让写作助手用擅长长文的模型、让代码助手用擅长编程的模型,互不干扰。见 [Agent 总览](../agents/overview.md)。

## 视觉与多模态

发图识图需要当前模型具备视觉能力;不具备时应用会提示。见[附件与图片](attachments-and-vision.md)。

## 中国大陆网络

设置中开启镜像源后,运行环境与依赖走国内镜像。模型服务本身的可达性取决于你选择的服务商。
