# 云同步与共享

Amadeus 默认纯本地。登录 Forsion 账号后,以下云能力**按需开启**:

## 云端库

在本地库之外,你可以创建**云端库**:内容存在云上,桌面端、[浏览器版、移动端](../reference/web-and-mobile.md)随处打开编辑。侧栏胶囊一键在"工作区(本地)"与"云端"之间切换,两边的会话与文件**互不混淆**。

## 目录级双向同步

不想整库上云?把本地库中的**某个目录**挂载到云端:

- 双向同步:本地改动推上去,云端(或其他设备)的改动拉回来;
- 重命名、移动都会正确跟随;
- 离线照常编辑,联网后自动对账;
- 同一文件两边都改过时,保留双方内容并生成**冲突副本**,不悄悄丢字。

> ⚠️ **注意**:云端单文件上限 5MB。超限文件会被跳过,侧栏以红色行明确提示,不会静默丢失。

## 页面共享(协作)

把某一页(连同其子页面)共享给其他 Forsion 用户:

- 生成**邀请链接**,可设访问密码;
- 成员按角色协作(编辑 / 查看);
- 页面上能看到协作者的**在线状态**;
- 共享页数量配额随订阅档位不同,以应用内显示为准。

## 公开分享

一键生成**公开链接**,任何人无需账号即可只读查看——发布一篇笔记就这么简单。链接可随时撤销,分享范围严格限定在你选的页面。

## 附件与云空间

云端库中的附件计入你的账号云存储,可在账号页统一查看。
