# 日历

日历空间把散落各处的"带日期的东西"聚合成一个视图。它自己不存数据——数据仍在你的表和待办里,日历只是它们的时间投影。

## 显式成员制

日历显示什么,由你**明确勾选**:

- **多维表**:任何带日期列的表都可以加入日历,行按日期落格;
- **待办**:笔记里的待办项按时间窗聚合展示;
- **Agent 日程**:[Muse 及其他 Agent](../agents/muse.md) 的计划任务以**只读**方式显示——一眼看到 AI 接下来要替你做什么。

没勾的不显示。日历不会自作主张把整个库翻个底朝天。

## 在日历里操作

- 点击条目直达来源(表中的行、笔记里的待办);
- 成员表的条目可以直接在日历上调整日期;
- Agent 日程条目只读——要改,去对应 Agent 的日程里改。

## 典型用法

把"项目任务表"(含截止日期列)加入日历,再让 Muse 每天早上把当天到期的事项发到[收件箱](../spaces/inbox.md)——看板、提醒、执行闭环。
