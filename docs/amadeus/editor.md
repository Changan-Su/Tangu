# 编辑器

Amadeus 编辑器是块式的:标题、段落、列表、表格、白板、代码……都是块,但书写体验保持纯 Markdown 的顺滑。

## Markdown 快捷输入

行首输入 Markdown 记号 + 空格,立即变成对应块:

| 输入 | 得到 |
|------|------|
| `#` ~ `######` + 空格 | 各级标题 |
| `-` 或 `*` + 空格 | 无序列表 |
| `1.` + 空格 | 有序列表 |
| `- [ ]` + 空格 | 待办项 |
| `>` + 空格 | 引用块 |
| ` ``` ` | 代码块 |
| `---` | 分割线 |

## 斜杠菜单

输入 `/` 呼出插入菜单,可以插入:

- **多维表** — 新建一张表并嵌入当前位置,见[多维表](databases.md);
- **白板** — 嵌入画布,见[白板](whiteboard.md);
- **分栏** — 把内容排成左右多栏;
- 代码块、引用、标题等常规块。

## 代码块

代码块带语法高亮,支持主流语言;右上角可复制、切换语言。

## 嵌入

`![[名字]]` 语法可以把其他内容嵌进笔记:

- `![[图片.png]]` — 内联图片(粘贴图片会自动保存并插入);
- `![[表名.db]]` — 嵌入一张多维表;
- `![[画布名.excalidraw]]` — 嵌入一块白板。

## 页面图标与封面

每篇笔记可以设置**图标**(emoji / 内置图标)和**封面图**,列表和页首都会展示——让常用笔记一眼可辨。
