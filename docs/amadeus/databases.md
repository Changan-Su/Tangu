# 多维表

多维表是 Amadeus 的结构化数据容器:管任务、追进度、记账、做清单。每张表是库里一个独立文件,可以嵌入任意笔记,也可以单独打开。

## 创建

- 笔记里输入 `/`,选择"数据库",当场插入一张新表;
- 已有的表用 `![[表名.db]]` 嵌到别处——**同一张表,处处同步**。

## 列类型

| 类型 | 用途 |
|------|------|
| 文本 | 任意文字,支持 `[[双链]]` |
| 数字 | 数值,可参与统计 |
| 勾选 | 是 / 否 |
| 日期 | 日期值,可上[日历](calendar.md) |
| 单选 / 多选 | 从选项池中选,选项带颜色 |
| 链接 | URL |

## 多视图

同一张表可以建多个**视图**,各自记住自己的筛选、排序与列配置:"全部任务"、"只看未完成"、"按截止日排序"来回切换,数据只有一份。

## 筛选、排序、统计

- **筛选**:按任意列的条件过滤行;
- **排序**:单列或多列排序;
- **统计**:列底部可显示汇总——计数、求和、平均等,记账表的合计一眼可见。

## 行详情

点开一行可以在弹层中查看和编辑整行字段,适合列多的表。

## 上日历

带日期列的表可以**加入日历**:在日历空间把这张表勾选为成员,行会按日期落在日历格子上,拖拽即改期。见[日历](calendar.md)。

## AI 也会用表

多维表对 AI 可见可写:"把这个待办表里逾期的行列出来"、"往读书表里加一行《三体》"都可以直接说。
