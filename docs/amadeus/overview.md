# Amadeus 总览

Amadeus 是 Forsion 内置的笔记空间。它的第一原则:**你的笔记是磁盘上的纯文本文件**,不是某个应用私有数据库里的记录。

## 一切都是文件

- 每篇笔记就是工作区目录里的一个 `.md` 文件;
- 目录树 = 文件夹结构,用 Finder / 资源管理器看到的和应用里一致;
- 格式与 **Obsidian 兼容**:双链、白板文件、属性都遵循通行格式,两边可以打开同一个库;
- 卸载应用、换电脑、用别的编辑器——笔记永远可用。

## 不只是文字

Amadeus 里这些内容都是"一等公民",各有专门的编辑体验:

| 内容 | 说明 |
|------|------|
| [笔记](editor.md) | 块式编辑器,Markdown 快捷输入 |
| [多维表](databases.md) | 类 Notion 的表格数据库,多视图 |
| [白板](whiteboard.md) | Excalidraw 兼容的无限画布 |
| [PDF](pdf-annotation.md) | 原生批注,写进 PDF 文件本身 |
| [日历](calendar.md) | 聚合表格、待办与 Agent 日程 |

## 与 AI 的联动

对话中的 AI 可以直接读写你的笔记(经[审批](../chat/tools-and-approvals.md)):"把刚才的讨论整理进会议笔记"、"根据这篇笔记列个行动清单"都是一句话的事。双链语法 `[[笔记名]]` 在聊天输入框里同样可用,把笔记精准递给 AI。

## 回收站

删除的笔记进入库内回收站,可以恢复;清空才真正删除。

## 置顶

常用的笔记可以置顶,在列表顶部常驻。

## 本地库与云端库

默认一切都在本地。登录后可以启用**云端库**,或把本地某个目录挂到云上双向同步——详见[云同步与共享](cloud-and-sharing.md)。
