# PDF 批注

把 PDF 放进库里,Amadeus 就是你的文献阅读器。与"批注存在应用数据库里"的方案不同,Forsion 的批注**直接写进 PDF 文件本身**——用系统预览、Adobe、手机上任何阅读器打开,批注都在。

## 支持的批注

**选中文字后:**

- 高亮(多种颜色)
- 下划线 / 删除线 / 波浪线

**任意位置:**

- **文字批注** — 直接在页面上写字;
- **便签** — 钉一条可展开的评论;
- **形状** — 矩形、椭圆、箭头等标记;
- **书签** — 把重要页加入书签列表,快速跳回。

## 阅读体验

- 连续滚动阅读、缩放;
- 批注列表侧栏,按页浏览所有标注,点击直达。

## 与 AI 配合

PDF 就在你的库里,AI 可以读:"总结这份论文的方法部分"、"把我在这份 PDF 里高亮过的句子整理成清单"。

> ⚠️ **注意**:批注写入的是文件本体。若 PDF 设置了禁止修改的权限,批注功能会受限。
