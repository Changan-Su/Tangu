# 浏览器版与移动端

Forsion 不把你锁在桌面前。登录同一个账号,浏览器和手机上就是同一套 Agent、同一份云端笔记。

## Forsion Web(浏览器版)

- 打开网页登录即用,无需安装;
- 运行在**云端模式**:对话、Agent、云端 Amadeus 库全部可用;
- 手机浏览器访问自动切换为单列布局。

## Forsion Mobile(Android)

- 安装 APK,默认连接云端;
- 与桌面同源的界面,针对手机优化:单列布局、手势导航;平板宽屏自动并排;
- 对话、Agent、云端笔记随身带。

## 平台能力对照

| 能力 | 桌面端 | Web / 移动端 |
|------|:---:|:---:|
| 对话 / Agent / 记忆 | ✅ | ✅ |
| 云端 Amadeus 库 | ✅ | ✅ |
| 本地笔记库 | ✅ | — |
| 本地文件 / 命令 / 桌面操作 | ✅ | — |
| 自动化 / 收件箱 / Muse 盯任务 | ✅ | 部分 |

一句话:**碰你电脑的能力只在你电脑上**,其余随处可用。

## 数据如何流转

Web 与移动端使用云端数据(云端库、云端会话);桌面端的本地内容除非你开启[云同步](../amadeus/cloud-and-sharing.md),否则不会出现在其他设备上。
