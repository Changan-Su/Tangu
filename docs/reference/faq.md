# 常见问题

## 连接与报错

**聊天区出现"⟳ 网络波动,X 秒后自动重试"是什么?**
瞬时网络故障,引擎正在自动重试(最多 3 次)。通常几秒内自愈,无需操作。

**"无法连接模型服务(fetch failed)"?**
网络不通或所选服务商不可达。检查网络、代理,以及 设置 → AI → 模型 里的服务商配置。

**"连接中途断开(terminated),回复未完成"?**
回复流到一半连接断了。为避免内容重复,这种情况不自动重试——点**重新生成**即可。

**"鉴权失败"?**
API Key 失效或账号登录过期。去 设置 → AI → 模型 重新配置或登录。

**"请求被限流 / 模型服务端错误"?**
服务商侧的临时问题,稍候重试;频繁出现可换用其他模型。

## 功能不可用

**模型列表是空的?**
还没有配置任何接入方式。见[模型与接入方式](../chat/models-and-providers.md)。

**发图片提示不支持?**
当前模型没有视觉能力,换一个带视觉的模型。

**微信 / 浏览器自动化 / 本地文件用不了?**
这些是本地能力,只在桌面端本地模式可用;Web / 移动端(云端模式)没有。见[平台能力对照](web-and-mobile.md)。

## 自动化

**规则到点没跑?**
两个最常见原因:

1. **应用没在运行** — 本地自动化由桌面端引擎执行,应用需保持运行(最小化即可);
2. **卡在审批** — 无人值守的任务等不到你点批准。给该规则的 Agent 配全自动权限,或只开放免审工具。见[自动化](../spaces/automation.md)。

## 数据

**笔记文件在哪?怎么备份?**
工作区目录 + `~/.forsion`,两个目录拷走即完整备份。见[数据与隐私](data-and-privacy.md)。

**删错了笔记?**
Amadeus 库内有回收站,先去那里找。

## 其他

**怎么看当前版本、查更新?**
设置 → 系统 → 关于,可查看版本与更新日志。

**问题不在列表里?**
直接问应用里的 AI(它了解 Forsion 的功能),或通过官网渠道反馈。
