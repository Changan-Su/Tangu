# 命令行(Tangu CLI)

Tangu 也活在终端里。对住在命令行里的人来说,这是最快的入口。

## 安装

在桌面端设置中一键安装 `tangu` 命令到系统 PATH。CLI 与桌面应用共用同一套程序与数据——**随应用自动更新**,Agent、配置、会话历史全部相通。

## 使用

```bash
# 交互式聊天(终端 UI)
tangu

# 非交互:直接执行一条指令,适合脚本与快捷键
tangu -e "把 ~/Downloads 里本周的截图归档到 Screenshots 文件夹"

# 安装引擎插件
tangu install npm:<包名>
```

## 与桌面端的关系

| | 桌面端 | CLI |
|---|--------|-----|
| Agent / 记忆 / 配置 | 同一套 | 同一套 |
| 审批 | 审批卡 | 终端内确认 |
| 适合 | 日常主界面 | 快速一嗓子、脚本集成、SSH 场景 |

## 自托管(进阶)

安装包内含 `tangu-server`,可以把引擎独立部署在服务器上供远程使用。面向进阶用户,此处不展开。

> 💡 **提示**:脚本中使用需要确认的命令(如插件安装)时,可加 `--yes` 跳过交互确认;无终端交互且未加 `--yes` 时命令会安全退出而不是擅自继续。
