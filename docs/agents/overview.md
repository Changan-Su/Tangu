# Agent 总览

在 Forsion 里,Agent 是**长期伙伴**:它有自己的人格、独立的记忆、专属的资料库和工作日志。你可以为不同的事创建不同的 Agent——写作教练、代码搭档、生活管家——它们互不串味。

## 一个 Agent 由什么组成

每个 Agent 在磁盘上就是一个文件夹,内容完全透明:

| 组成 | 说明 |
|------|------|
| **配置** | 名字、头像、使用的模型、开放哪些工具、权限档位 |
| **人格(SOUL)** | 一份 Markdown:它是谁、什么口吻、什么原则。写得越具体,角色越立体 |
| **记忆(MEMORY)** | 相处中记下的你的偏好与事实,见[记忆系统](memory.md) |
| **资料库(Library)** | 你塞给它的参考资料,回答时可随时查阅 |
| **日志(LOG)** | 它替你干过什么,有账可查 |

## 创建与管理

- **界面创建**:在 Agent 管理面板新建,填名字、写人格、选模型即可;
- **让 AI 代劳**:直接在对话里说"帮我建一个每天督促我锻炼的教练 Agent",AI 会调用管理工具替你建好;
- **随时调整**:模型、工具开关、人格文件都可事后修改,立即生效。

## 内置 Agent

- **默认助手** — 通用型,开箱即用;
- **[Muse](muse.md)** — 主动型助理,会盯任务、感知你的使用情境、主动找你。

## 每个 Agent 各用各的配置

模型、[工具黑白名单](../chat/tools-and-approvals.md)、权限档位都是按 Agent 独立设定的。让"文件整理员"只能动文件、"资讯官"只能联网,权责分明。

## 云端 Agent

登录 Forsion 账号后,你的 Agent 也可以在云端使用——[浏览器版和移动端](../reference/web-and-mobile.md)上同样能与它们对话。
