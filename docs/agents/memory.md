# 记忆系统

好的助手不用每次重新自我介绍。Forsion 的 Agent 会在对话中**自动沉淀记忆**:你的称呼、口味、工作习惯、项目背景……下次对话直接带着这些理解上场。

## 记忆长什么样

**每个 Agent 的记忆彼此独立**,存放在它自己的文件夹里,就是一份 Markdown 文件:

- 想看它记了什么?打开看;
- 记错了?直接改;
- 想让它忘掉?删掉那一行。

没有黑盒。记忆完全归你掌控。

## 记忆 vs 资料库

| | 记忆(MEMORY) | 资料库(Library) |
|---|---|---|
| 谁来写 | Agent 在相处中自动记 | 你主动放进去 |
| 内容 | 关于你的偏好与事实 | 参考资料、文档、规范 |
| 典型例子 | "用户喜欢简洁的回答" | 一份产品需求文档 |

## 云同步(可选)

登录 Forsion 账号后可以把记忆同步到云端,让[浏览器版与移动端](../reference/web-and-mobile.md)上的同一个 Agent 记得同样的事。同步是**本地优先**的:本地文件是主本,云端是副本,不开同步就完全不上传。
