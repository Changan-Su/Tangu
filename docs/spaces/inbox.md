# 收件箱

当 AI 开始主动干活([盯任务](../agents/muse.md)、[自动化](automation.md)、后台任务),它的产出需要一个不会被错过的落点。收件箱就是这个落点:**所有 Agent 主动发给你的消息,统一在这里**。

## 会收到什么

- Muse 盯任务的提醒与报告("你要的日程摘要来了");
- 自动化规则的执行结果;
- 后台长任务完成的通知;
- 任何 Agent 在工作中觉得该告诉你的事。

## 使用

- 未读消息有醒目标记,空间图标带角标;
- 点开消息可以**跳到来源**——哪条规则、哪个会话产生的,一键回溯;
- 处理完的消息可删除,列表保持干净。

## 与系统通知的关系

重要消息同时走系统通知(即时弹出)与收件箱(可回溯的清单)。错过了弹窗没关系,收件箱里都在。
