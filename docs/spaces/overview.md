# Space 总览

Space(空间)是 Forsion 组织功能的方式:每个空间是一个独立的功能界面,专注做好一类事。侧栏即空间清单,点击切换。

## 内置空间

| 空间 | 干什么 |
|------|--------|
| **对话** | 与 Agent 聊天,一切的起点 |
| **Amadeus** | [笔记系统](../amadeus/overview.md):文档、多维表、白板、PDF |
| **日历** | [聚合一切带日期的东西](../amadeus/calendar.md) |
| **收件箱** | [Agent 主动发给你的消息](inbox.md) |
| **自动化** | [让 AI 按规则自己干活](automation.md) |
| **编码** | [写代码、当场预览](coding.md) |

## 启用与关闭

用不到的空间可以关掉,界面更清爽:设置 → 选项 → 空间。随时可以重新打开,数据不受影响。

## 从市场安装空间

[应用市场](../customization/market.md)的"空间"分类提供第三方空间——安装后立即出现在侧栏,无需重启。每个空间自带说明,不合适就卸载。

## 空间与工作区布局

空间内容以标签页打开,支持拖拽分屏:左边笔记、右边对话,一边看资料一边让 AI 干活。还可以把任意内容拆成独立窗口或置顶 Mini 卡片。见[核心概念](../getting-started/concepts.md)。
