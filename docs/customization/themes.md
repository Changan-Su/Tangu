# 主题与外观

外观设置在 设置 → 选项 → 外观。Forsion 的主题体系分三个独立维度,自由组合:

## 设计语言

决定应用的**气质**:字体、密度、质感。多套内置设计语言各有性格——有的书卷气,有的现代利落;部分设计语言还自带成套配色。

## 配色

决定应用的**颜色**。内置多套配色(Genesis、经典、珊瑚、柔青、薰衣草……),另有**自定义**:

- **强调色** — 按钮、选中态、链接的主色;
- **背景色** — 整体底色的基调。

两个自由度独立可调,想要"米白纸面 + 黛青强调"这种个人配方,自定义里直接拾色。

## 明暗模式

浅色 / 深色 / 跟随系统。所有设计语言与配色都同时适配明暗两态。

## 主题包

- **市场安装**:[应用市场](market.md)的"主题"分类,一键安装即用;
- **手动安装**:主题包是数据目录 `themes/` 下的一个文件夹,拿到喜欢的主题包放进去即可;
- 主题作者可以通过市场**投稿**自己的作品。

## 全局一致

主题作用于整个应用——对话、Amadeus 笔记、各空间、弹窗,一套气质贯穿到底。
