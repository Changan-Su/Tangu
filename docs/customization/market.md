# 应用市场

应用市场是 Forsion 生态的门面:别人打磨好的能力,你一键装上。

## 六类内容

| 分类 | 装到哪 |
|------|--------|
| **技能** | 给 AI 的[做事说明书](../agents/skills.md) |
| **代理** | 现成的 [Agent](../agents/overview.md):人格、配置一步到位 |
| **插件** | [引擎插件](plugins.md),给 AI 添工具 |
| **空间** | 新的[功能空间](../spaces/overview.md) |
| **主题** | [外观主题包](themes.md) |
| **笔记插件** | Amadeus 专属增强 |

## 安装与更新

- 详情页看介绍、截图与依赖,一键安装;
- **更新**页集中列出已装内容的可用更新;
- 上架内容经过审核,安装的始终是**过审版本**——上游仓库事后的改动不会悄悄进入你的机器。

## 投稿

做了个好东西?市场内直接**投稿**:提交你的 GitHub 仓库,通过审核后上架,全体用户可见。
