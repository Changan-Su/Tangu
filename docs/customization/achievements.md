# 成就系统

Forsion 功能很多,成就系统是一张**探索地图**:第一次用某个功能、达成某个里程碑,就会解锁对应成就。

## 怎么玩

- 正常使用即可,达成瞬间会有**成就弹窗**;
- 打开**成就面板**查看全部成就:亮着的是走过的路,灰着的是还没去过的地方;
- 解锁后**领取**成就,积累点数;
- 还有不在列表里的**隐藏成就**,留给爱折腾的人。

## 把成就当学习清单

灰色成就往往对应一个你还没用过的功能——看到"创建第一条自动化规则"没点亮?正好去[自动化空间](../spaces/automation.md)试一把。跟着成就走一圈,主要功能就都摸过了。
