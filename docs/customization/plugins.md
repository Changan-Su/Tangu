# 插件

Forsion 的插件生态开放且分工明确,共两类:

| 类型 | 扩展什么 | 例子 |
|------|----------|------|
| **Forsion 插件**(桌面) | 界面与笔记能力:新面板、笔记增强、数据看板 | 全机使用时间统计 |
| **Tangu 插件**(引擎) | AI 的工具箱:给 Agent 添加新工具 | Computer Use(让 AI 看屏幕、操作桌面应用) |

## 安装渠道

- **应用市场**(推荐):一键安装,更新统一管理,上架内容经过审核且**锁定过审版本**;
- **GitHub 仓库**:输入仓库地址直接安装市场外的插件;
- **命令行**(引擎插件):`tangu install npm:<包名>` 从 npm 安装,见[命令行](../reference/cli.md)。

插件如果依赖某个本机应用,详情页会列出并提供**一键安装**引导。

## 插件设置

带设置项的插件在其**详情页**提供表单,改完即生效,无需改配置文件。

## 安全模型

插件能力大,约束也硬:

- npm 渠道安装**只下载文件、绝不执行安装脚本**,杜绝装包即中招;
- 引擎插件提供的工具**默认纳入审批体系**——插件自己也无法绕过"动手前问你";
- 所有插件安装在数据目录 `plugins/` 下,一插件一文件夹,看得见、删得掉。

## 开发者

插件用 JavaScript/TypeScript 编写,官方提供类型定义与模板仓库。开发文档在筹备中;现阶段可参考市场上的开源插件源码。
