## 1.3.1 (2026-06-25)

- 新增「应用内更新」:设置 → 关于 可一键检查更新;检测到新版会在主界面顶部提示
  - Windows / Linux:可直接在应用内下载并重启安装,无需手动去下载页
  - macOS:检测到新版后引导前往 Releases 页手动下载(暂未启用 mac 应用内自动安装)
  - 启动时静默检查一次;下载与安装均需你点击确认

## 1.3.0 (2026-06-25)

- 新增「外部 Agent 引擎」:新会话开始处可选用 Claude Code、Codex 等第三方 AI Agent 框架代替内置 Tangu 引擎(经 ACP 接入)——
  - 像用 Tangu 一样在主界面对话,自动适配该引擎的模型选择与 slash 命令
  - 设置 → Agent CLIs:查看本机已检测到的引擎,并为每个引擎设默认模型
  - 引擎选择器仅在对应引擎「已检测到」时出现;一个会话从始至终使用同一种引擎
- 新增「群聊模式」:让多个智能体围绕同一话题轮流发言、投票,并可由主持人总结(本地工作区)
- 新增「官方账号订阅登录」:用你的 Claude / ChatGPT 订阅额度跑 Tangu(OAuth 登录,无需 API Key)
- 记忆改为「本地优先」:记忆与日志默认存在本机(~/.tangu),可在设置里与 Forsion 云端双向同步(默认手动,隐私优先)
- 代码编辑更稳:新增结构化补丁(apply_patch)与本机文件操作安全闸(路径越界需确认)
- 修复:已安装的 Claude Code / Codex 偶尔检测不到;云端会话不再误列本地模型、也不再显示外部引擎(外部引擎仅本地会话可用,且与群聊互斥)

## 1.2.0 (2026-06-21)

- 新增「微信远程」:在 设置 → 微信远程 扫码连接你的微信 bot,从微信里直接和 Tangu Agent 对话——
  - 作为一个独立的工作区(Project)出现在侧栏顶部,可折叠展开它下面的会话
  - 微信里可用 `/new` 新建会话、`/list` 列出、`/switch <序号>` 切换、`/help` 帮助;回复「停止」中止任务、「批准 / 拒绝」处理待批操作;有「正在输入」状态
  - Agent 可把工作区里的文件 / 图片直接发回到你的微信(「把这个文件发我微信」)
  - 主界面用于设置 / 连接,可查看与切换「正在连接的会话」
- 新增「浏览器工具」:Agent 可用本地轻量浏览器实时搜索网页并交互(`browser_search` 等),默认 DuckDuckGo,支持 Bing / Google / 百度
- 自定义 Provider 新增「拉取模型」:填了 Base URL / API Key 后一键拉取该端点可用模型列表,搜索勾选即可
- 侧栏改进:微信连接默认置顶;工作区可拖拽调整排序;新增会话搜索;删除以「归档」为主操作(永久删除收进归档区)
- 聊天区新增「划线引用」:选中文字浮出「引用」,作为引用块拼到下一条消息前(类 ChatGPT)
- 顶栏精简:去掉一排状态胶囊(Host / 微信 / 在线 等),只在掉线时提示
- 更顺手:微信来消息、开关后台智能体等变化现在自动呈现,无需手动刷新页面
- ⚠️ 「后台智能体」Historian(历史员)与 Muse(缪斯)仍为**实验性功能,目前极其不稳定**,可能出现卡住、重复触发、消耗异常等问题;默认关闭,请谨慎在非重要工作区试用,正式使用前建议保持关闭

## 1.1.0 (2026-06-18)

- 新增「上下文用量条」:输入框下方实时显示当前模型上下文占用百分比与本会话 token 消耗,一眼看清还剩多少额度
- 新增「压缩上下文」:对话变长时一键(输入栏按钮或 `/compact`)总结后精简续接;接近上限会自动压缩,避免超长报错、长对话也能稳稳继续
- 新增「智能体」(Normal Agent):在 设置 → 智能体 里创建可复用的对话人格(系统提示词 + 模型 + 设置),用 `/agent` 一键切换;也可让 AI 在对话中自己沉淀出新的智能体
- 新增「后台智能体」(默认关闭,在 设置 → 后台智能体 开启并选模型;记录隔离,不出现在会话列表):
  - Historian(历史员):对话进行中自动总结会话标题,并在需要时维护你的长期记忆与日志
  - Muse(缪斯):后台持续思考「现在能为你做点什么」,产出 TODO 清单;可多选任务、选一个会话一键注入并开始执行
- 侧栏新增「后台智能体」入口:进入 Historian / Muse 工作区,查看它们的活动与 TODO

## 1.0.3 (2026-06-17)

- 修复:本地工作区的文件操作(预览、打开目录、重命名、删除、在文件管理器中显示、拖出)全部失效、点击即报错的问题
- 新增:设置 → 高级「导出会话日志」,将当前会话的全部对话与后端运行日志打包为一个 JSON 文件,便于反馈与排障
- macOS:左上角 Logo 与「Tangu」字样右移,不再被窗口左上角的红绿灯按钮遮挡
- Windows:移除顶部默认菜单栏(文件/编辑/视图/窗口),与 macOS 观感统一
- 新增 `/loop <轮数>` 指令调节单次对话的最大循环轮数(默认上限提升至 90 轮),输入栏与 TUI 均可用
- 循环耗尽(到达轮数上限仍未完成)时给出明确提示,可发送「继续」或用 `/loop` 调高上限

## 1.0.2 (2026-06-16)

- 修复:全新安装后云端地址默认连到本地(localhost),现在内置默认指向生产环境 api.forsion.net
- 自建/私有部署仍可通过 ~/.tangu/.env 的 TANGU_CLOUD_URL 覆盖云端地址
