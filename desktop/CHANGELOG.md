## 1.0.3 (2026-06-17)

- 修复:本地工作区的文件操作(预览、打开目录、重命名、删除、在文件管理器中显示、拖出)全部失效、点击即报错的问题
- 新增:设置 → 高级「导出会话日志」,将当前会话的全部对话与后端运行日志打包为一个 JSON 文件,便于反馈与排障
- macOS:左上角 Logo 与「Tangu」字样右移,不再被窗口左上角的红绿灯按钮遮挡
- Windows:移除顶部默认菜单栏(文件/编辑/视图/窗口),与 macOS 观感统一
- 新增 `/loop <轮数>` 指令调节单次对话的最大循环轮数(默认上限提升至 90 轮),输入栏与 TUI 均可用
- 循环耗尽(到达轮数上限仍未完成)时给出明确提示,可发送「继续」或用 `/loop` 调高上限

## 1.0.2 (2026-06-16)

- 修复:全新安装后云端地址默认连到本地(localhost),现在内置默认指向生产环境 api.forsion.net
- 自建/私有部署仍可通过 ~/.tangu/.env 的 TANGU_CLOUD_URL 覆盖云端地址

## 1.0.1 (2026-06-15)

- 本地图片识别:AI 可查看工作区里或拖入的图片(view_image 工具),聊天内显示图片缩略图
- 新增开发者模式:在「关于」连点版本号 10 次解锁,可编辑 Forsion 云端地址、重新进入引导
- 首启引导新增「选择主题」与「默认本地文件夹」两步;引导不再要求填写云端地址
- 默认主题改为 Qbird 浅色;语言与深浅模式切换移至右上角
- 账号卡片改用统一用户卡片样式,正确显示头像与会员等级

## 1.0.0 (2026-06-15)

- 新增中英文界面切换(默认中文),侧栏左下角与设置内均可切换
- 设置新增「关于」页:版本号与更新日志
- Forsion 账号登录移至侧栏左下角(头像/昵称,点击进入个人中心);不登录也能正常使用
- 模型 / Provider 在设置与模型选择器中按 Provider 分组,默认折叠
- 工作区支持重命名 / 新建文件夹 / 在文件管理器中显示 / 删除到回收站 / 拖出
- 修复:工具调用偶发结束循环;流式输出时上滑不再被强制拽回底部
