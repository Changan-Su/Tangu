# Forsion 更新日志

## 2.6.5 (2026-07-19)

## Forsion 2.6.0 (2026-07-16)

- **内置白板**——命令面板/笔记库侧栏「新建白板」即建即画:手绘图形、连线、文字、素材库、明暗跟随主题;白板以开放的 `.excalidraw.md` 格式存在 vault 里(与支持该格式的白板工具互通),笔记里 `[[白板名.excalidraw]]` 一点直达,树里有专属图标、绝不会被当成普通笔记误开
- **PDF 原生批注**——vault 里的 PDF 应用内直接批注:高亮 / 下划线 / 删除线 / 波浪线 / 便签 / 文字 / 形状 / 书签,批注**写进 PDF 文件本身**(用其他阅读器打开同样可见);带缩略图侧栏与批注列表,选中文字即弹批注条
- **本地笔记按条目上云**——不必整库上云:本地库里任选文件夹或单篇笔记「同步到云端」,即挂到你的云端库(双向同步、离线可用、改名/移动都跟得上);正文里的图片附件一并带走;云端附件在 账户 → 云空间 可见可下载(单文件上限 5MB,超限会在侧栏明确提示)
- **全新「自动化」空间**——规则 = 触发器(定时 / 事件)+ 指定任意智能体无人值守执行,三栏管理:规则列表 / 流程构建器 / 触发记录;智能体还有了自己的**日程表**(纯规划提醒,或到点自动执行;每条规则的执行转录可回看)
- **成就系统**——日常使用即解锁徽章(聊天、笔记、主题、微信远程……),达成时输入框上方弹出动画,左栏奖杯图标进成就面板**领取积分**;插件也能注册自己的成就
- **语音输入**——聊天输入框新增麦克风:可选**本地离线转写**(一键下载模型,之后不联网)或云端 / 自带 provider 模型转写;macOS 首次使用会请求麦克风权限
- **群聊记得之前聊了什么**——聊到一半开群聊,参与者现在能看到此前的对话内容(先自动压缩再注入,长对话也不超载);群聊里的后续发言同样延续上下文
- **自带 API Key(BYOK)体验修复三连**——① 添加直连 provider 后,模型选择器按 provider 分组稳定列出你的模型(不再被云端同名模型顶掉);② **不登录 Forsion 也能用自己的 key 正常聊天**(此前会报云端 401);③ 官方 OpenAI 新一代模型(gpt-5.x)适配:普通对话与思考模式都可用、**思考过程可见**,不再报「function tools not supported」
- **默认思考深度改为「中」**——新会话默认带思考推理(模型选择器里可随时调档或关闭);云端会话的模型菜单会说明为何只列托管模型
- **每个智能体可配工具黑白名单**——智能体编辑页按需允许/禁用内置工具;**Muse 可以「盯任务」**:设定时/事件规则,到点自动唤起(以本地活动日志为数据源,设置可关)
- **Tangu 引擎插件支持从 npm 安装**——`tangu install npm:<包名>`:直接拉取包内容、**不执行任何安装脚本**,校验完整性后原子落位,更安全;插件工具可自声明审批档位
- **Android 版(预览)**——手机端 App 随桌面版本同步出包(默认连云端,也可本地模式);网页/云端会话现在也能**创建自定义智能体**(与桌面同步)
- 模型网络抖动自动重试时,界面显示「第 N / M 次重试」而不是干等;运行错误信息更易读
- **Calendar Space 全面重做**——不再需要「日历日期」「待办」这两个特殊属性:现在任何数据库,只要有一个**日期属性**,就能通过右栏**「+ 添加 Forsion database」**、或数据库**视图设置 →「添加到 Calendar Space」**加入日历;加入时指定哪一列作日期锚点、可选哪一列作完成勾选(都能随时改)。右栏日历列表里每个库都能**重命名 / 设为默认 / 在新标签打开 / 改列映射 / 改颜色 / 移出日历**。旧的日历数据库会**自动迁移**,无需手动重加
- **数据库工具栏增强**——数据库右上角新增**搜索**(按标题过滤行)、**在新标签打开为整页**、**视图设置**齿轮;视图设置里补上**排序**与**筛选**入口
- **待办清单时间窗 + 设置**——待办清单顶部可切**日 / 3日 / 周 / 月 / 自定义天数**(以今天为中心前后对称),自动只显示该时间段内的待办;⚙ 设置里可**隐藏已完成**、按**字母 / 未完成优先 / 已完成优先**排序
- **全局快速查找**——Ribbon 新增搜索图标(或 ⌘P)呼出居中面板,按名称快速跳转到**笔记 / 数据库 / 聊天会话**,没输入时列出最近;右栏「添加 Forsion database」也改成同款居中搜索面板
- **笔记标题一键进正文**——在标题里按**回车**、或光标在标题**末尾按 →**,直接跳到正文第一行
- **修复**——事件卡里的完成勾选框在深/浅色主题下的显示适配
- **独立窗口(多窗口工作区)**——把任意标签页**拖出主窗口**、或在标签上**右键「移到新窗口」**,它就变成一个独立窗口(默认占满主区);可以再把别的标签页**跨窗口拖进来**并排成多栏。独立窗口**没有左侧活动栏**(与主窗口区分),窗口的位置 / 大小 / 里面的布局都会记住,**下次开 App 自动恢复**
- **Mini 悬浮卡片模式(带快捷键)**——按 **⌘/Ctrl+Shift+M**(或命令面板「打开 Mini 卡片」)弹出一个小巧的悬浮卡片:固定 **3:4 竖版**比例、顶部是 Space 切换条、下面直接是内容(基于移动端单列界面精简,去掉多余分区),**始终置顶、与主窗口并存**;把卡片**拖到屏幕边缘会自动吸附折叠**成一条窄边,**鼠标移上去再展开**

- **修复一批"点了没反应"的菜单**——新建文件夹、新建 Base、重命名文件夹、书签、存为集合、添加属性(桌面端不支持浏览器 prompt 弹窗,全部换成应用内输入框);「嵌入块引用」剪贴板没有引用时现在会弹输入框而不是静默无反应;「链接数据库」列表不再偶发为空

- **撤销 / 重做(Cmd+Z / Cmd+Shift+Z / Cmd+Y)**——笔记编辑支持文档级撤销:不仅块内文字,块的新建 / 删除 / 合并 / 移动 / 斜杠转换都能一步步撤销和重做(切到别的笔记后历史独立,不会误改到其它笔记)

- **聊天里你发的消息也显示头像与昵称**(取 Forsion 账户信息,与助手左右镜像)

## 1.5.0 (2026-06-27)

- 界面全面贴近 Forsion 统一设计语言(LCL)预览,并升级为**可停靠工作台**:
  - **聊天外壳重做**:工具调用、审批、待办、询问等卡片统一为「中性柔影浮卡」,输入框浮起,按钮 / 胶囊 / 图标钮对齐预览;整体更轻盈、留白更舒展(头像与原信息保留)
  - **可停靠工作台**:**左侧「工作会话」、中间对话、右侧「文件 / 目录 / 记忆 / 子聊天」全部纳入工作台**,每个面板都是带**图标 + 名称**、可**自由拖拽、调宽、成组 / 分离、停靠**的独立卡片(拖拽更丝滑);布局自动记忆,重启还原;顶栏「侧栏 / 右栏」按钮整体收起 / 展开;打开「设置 / 引导」时面板自动收起,设置铺满全屏
  - **更灵动**:展开会话列表、弹出菜单等多处加入微动效(统一「切换 Agent」那种轻盈手感);尊重系统「减少动态效果」设置
  - **主题升级为「设计语言 × 配色」双轴**(对齐 Amadeus / LCL):**设计语言**(Lovable 平展纸感 / Soft 柔影浮卡)决定结构(圆角 / 字体 / 浮卡布局);**配色**(奶油 / 珊瑚 / 柔青 / 薰衣草 / 自定义)只决定颜色——二者正交,可自由组合(如「柔青 + 柔影浮卡」)。Soft 语言下侧栏 / 主区成为渐变舞台上的圆角浮卡(角落辉光、柔影、无硬边框)
  - **默认黄金分割布局**:中间对话约占 62%、左右面板各约 19%,更聚焦;面板大小 / 位置仍自动记忆,重启还原
  - 面板 tab **更紧凑圆润**(收窄留白、圆角加大)
  - 明暗 / 扁平 / 玻璃开关在所有 设计语言 × 配色 上一致生效
- 旧主题**自动迁移**:Lovable→奶油、Echo→珊瑚、Qbird→柔青、Dreamer→Soft + 薰衣草;升级后可在 设置 → 主题 自由组合设计语言与配色

## 1.3.0 (2026-06-25)

- 新增「外部 Agent 引擎」:新会话开始处可选用 Claude Code、Codex 等第三方 AI Agent 框架代替内置 Tangu 引擎(经 ACP 接入)——
  - 像用 Tangu 一样在主界面对话,自动适配该引擎的模型选择与 slash 命令
  - 设置 → Agent CLIs:查看本机已检测到的引擎,并为每个引擎设默认模型
  - 引擎选择器仅在对应引擎「已检测到」时出现;一个会话从始至终使用同一种引擎
- 新增「群聊模式」:让多个智能体围绕同一话题轮流发言、投票,并可由主持人总结(本地工作区)
- 新增「官方账号订阅登录」:用你的 Claude / ChatGPT 订阅额度跑 Tangu(OAuth 登录,无需 API Key)
- 记忆改为「本地优先」:记忆与日志默认存在本机(~/.tangu),可在设置里与 Forsion 云端双向同步(默认手动,隐私优先)
- 代码编辑更稳:新增结构化补丁(apply_patch)与本机文件操作安全闸(路径越界需确认)
- 修复:已安装的 Claude Code / Codex 偶尔检测不到;云端会话不再误列本地模型、也不再显示外部引擎(外部引擎仅本地会话可用,且与群聊互斥)

## 1.2.0 (2026-06-21)

- 新增「微信远程」:在 设置 → 微信远程 扫码连接你的微信 bot,从微信里直接和 Tangu Agent 对话——
  - 作为一个独立的工作区(Project)出现在侧栏顶部,可折叠展开它下面的会话
  - 微信里可用 `/new` 新建会话、`/list` 列出、`/switch <序号>` 切换、`/help` 帮助;回复「停止」中止任务、「批准 / 拒绝」处理待批操作;有「正在输入」状态
  - Agent 可把工作区里的文件 / 图片直接发回到你的微信(「把这个文件发我微信」)
  - 主界面用于设置 / 连接,可查看与切换「正在连接的会话」
- 新增「浏览器工具」:Agent 可用本地轻量浏览器实时搜索网页并交互(`browser_search` 等),默认 DuckDuckGo,支持 Bing / Google / 百度
- 自定义 Provider 新增「拉取模型」:填了 Base URL / API Key 后一键拉取该端点可用模型列表,搜索勾选即可
- 侧栏改进:微信连接默认置顶;工作区可拖拽调整排序;新增会话搜索;删除以「归档」为主操作(永久删除收进归档区)
- 聊天区新增「划线引用」:选中文字浮出「引用」,作为引用块拼到下一条消息前(类 ChatGPT)
- 顶栏精简:去掉一排状态胶囊(Host / 微信 / 在线 等),只在掉线时提示
- 更顺手:微信来消息、开关后台智能体等变化现在自动呈现,无需手动刷新页面
- ⚠️ 「后台智能体」Historian(历史员)与 Muse(缪斯)仍为**实验性功能,目前极其不稳定**,可能出现卡住、重复触发、消耗异常等问题;默认关闭,请谨慎在非重要工作区试用,正式使用前建议保持关闭

## 1.1.0 (2026-06-18)

- 新增「上下文用量条」:输入框下方实时显示当前模型上下文占用百分比与本会话 token 消耗,一眼看清还剩多少额度
- 新增「压缩上下文」:对话变长时一键(输入栏按钮或 `/compact`)总结后精简续接;接近上限会自动压缩,避免超长报错、长对话也能稳稳继续
- 新增「智能体」(Normal Agent):在 设置 → 智能体 里创建可复用的对话人格(系统提示词 + 模型 + 设置),用 `/agent` 一键切换;也可让 AI 在对话中自己沉淀出新的智能体
- 新增「后台智能体」(默认关闭,在 设置 → 后台智能体 开启并选模型;记录隔离,不出现在会话列表):
  - Historian(历史员):对话进行中自动总结会话标题,并在需要时维护你的长期记忆与日志
  - Muse(缪斯):后台持续思考「现在能为你做点什么」,产出 TODO 清单;可多选任务、选一个会话一键注入并开始执行
- 侧栏新增「后台智能体」入口:进入 Historian / Muse 工作区,查看它们的活动与 TODO
