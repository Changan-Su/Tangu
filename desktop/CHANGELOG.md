## Tangu 2.0.0 (2026-06-28)

- **全新 2.0:桌面端在全新引擎上重建**——整个界面改用「可停靠工作台」引擎(类 Obsidian / VS Code),面板拖拽 / 分屏 / 调宽更顺滑,布局自动记忆、重启还原;原有功能(对话、群聊、微信、主题、智能体、记忆等)全部沿用
- **顶部原生标签栏 + 圆角胶囊标签**:一条标签栏 = 左侧视图图标 ｜ 中间会话标签 ｜ 右侧视图图标,均可拖拽移动 / 分屏;激活标签为浅底圆角胶囊;标签栏末尾「＋」一键新建
- **新建标签页 = 空白启动器**:点「＋」开一个空白页,按「主区 / 侧栏视图」分组列出全部视图(新对话 / 微信 / 后台智能体 / 工作区 / 目录 / 记忆 / 子聊天),选中即在对应区打开
- **左侧功能条**:新建、主题、明暗、中英、反馈、命令面板 竖排在最左细条,随手可点
- **左右栏折叠钮移到面板边缘**:收起侧栏后按钮仍在原处,随时重新展开
- **设置界面重构为四大类**:选项 / AI / 核心插件 / 社区插件;返回与搜索常驻顶部(macOS 自动避让交通灯按钮);「常规设置」把连接与 Forsion 账户合并到同屏
- **快捷键自定义**:设置 → 快捷键 可为命令面板及所有命令录制 / 改键 / 解绑 / 一键重置,组合键冲突自动避让
- **默认智能体更名「Tangu Arioso」**(原 Xyra,记忆 / 日志无缝沿用)并内置默认头像;设置 → 智能体 可删除已设头像(头像意外丢失会自动补回默认,除非你主动删过)
- **启动闪屏**:打开到界面就绪之间用「神树」动画过渡,消除白屏
- 关于页更新日志支持 Markdown 行内格式(代码 / 加粗 / 链接正常显示)


## Tangu Go (2026-06-27)
- **聊天里你发的消息也显示头像与昵称**(取 Forsion 账户信息,与助手左右镜像)
- **工具调用更清爽**:连续的工具调用聚合成一张可折叠卡片(参考 Codex)——折叠时显示「正在写入 / 编辑…」或完成摘要(如「写入 1 文件 · 编辑 3 文件 · 运行 2 命令」),展开看每步的动作、目标与增删行数
- **右栏「记忆」改为「当前 Agent 记忆」**:随会话 / 群聊参与者实时切换,显示该智能体的记忆与最近几天日志(可折叠);单智能体时可直接追加记忆
- **目录与浮动目录**:右栏「目录」按对话轮次 + 标题成列;正文左侧新增悬浮目录条(hover 展开、点击跳转),长对话也能快速定位
- **文件面板升级为工作区文件夹树**:列出本地工作区并逐级展开真实磁盘目录、预览文件;会话与文件面板手风琴联动(进入某工作区即展开它、收起其余,两侧同步)

## 1.6.0 (2026-06-27)

- **主题支持「拖入安装」**(类 Obsidian):把主题文件夹放进 `~/.tangu/themes/<名字>/`(内含 `theme.json` + `theme.css`)即可装第三方设计语言;设置 → 主题 新增「**打开主题文件夹**」「**重载主题**」两个按钮——放好文件点「重载主题」就能立即看到并选用,无需重启
- **Soft 柔影**改为「拖入式示例主题」:首次启动自动种到 `~/.tangu/themes/soft/`,既可直接用,也能照着它改色 / 改圆角做出自己的主题(Lovable 仍内置为默认基底,始终可用)

## 1.5.0 (2026-06-27)

- 界面全面贴近 Forsion 统一设计语言(LCL)预览,并升级为**可停靠工作台**:
  - **聊天外壳重做**:工具调用、审批、待办、询问等卡片统一为「中性柔影浮卡」,输入框浮起,按钮 / 胶囊 / 图标钮对齐预览;整体更轻盈、留白更舒展(头像与原信息保留)
  - **可停靠工作台**(类 Obsidian / VS Code):**左侧「工作会话」、中间对话、右侧「文件 / 目录 / 记忆 / 子聊天」全部纳入工作台**,每个面板都是带**图标 + 名称**、可**自由拖拽、调宽、成组 / 分离、停靠**的独立卡片(拖拽更丝滑);布局自动记忆,重启还原;顶栏「侧栏 / 右栏」按钮整体收起 / 展开;打开「设置 / 引导」时面板自动收起,设置铺满全屏
  - **更灵动**:展开会话列表、弹出菜单等多处加入微动效(统一「切换 Agent」那种轻盈手感);尊重系统「减少动态效果」设置
  - **主题升级为「设计语言 × 配色」双轴**(对齐 Amadeus / LCL):**设计语言**(Lovable 平展纸感 / Soft 柔影浮卡)决定结构(圆角 / 字体 / 浮卡布局);**配色**(奶油 / 珊瑚 / 柔青 / 薰衣草 / 自定义)只决定颜色——二者正交,可自由组合(如「柔青 + 柔影浮卡」)。Soft 语言下侧栏 / 主区成为渐变舞台上的圆角浮卡(角落辉光、柔影、无硬边框)
  - **默认黄金分割布局**:中间对话约占 62%、左右面板各约 19%,更聚焦;面板大小 / 位置仍自动记忆,重启还原
  - 面板 tab **更紧凑圆润**(收窄留白、圆角加大)
  - 明暗 / 扁平 / 玻璃开关在所有 设计语言 × 配色 上一致生效
- 旧主题**自动迁移**:Lovable→奶油、Echo→珊瑚、Qbird→柔青、Dreamer→Soft + 薰衣草;升级后可在 设置 → 主题 自由组合设计语言与配色
- 修复:**个人中心**(账户页)里可编辑卡片在内容过长时会把「保存」按钮顶出视口——文本框改为内部滚动,按钮常驻

## 1.4.0 (2026-06-26)

- 主题系统升级到 Forsion 全家桶统一设计语言(LCL):
  - 新皮肤 **Lovable**(奶油纸感 · 单色,默认)、**Echo**(珊瑚 / 薰衣草 · 圆润)、**QBird**(石墨 · 柔青)、**Dreamer · 梦想家**(薰衣草 / 暖桃 · 圆润,渐变云霞底 + 柔影浮卡,与 Amadeus 同源)
  - 新增「**自定义**」取色皮肤:任选强调色,背景自动微染、明暗自适应
  - 聊天里的**工具调用卡片**改为统一的整框 + 柔阴影浮卡(对齐 LCL 预览),换肤 / 明暗 / 扁平一致跟随
  - 新增「**扁平 / 立体**」开关:一键去除卡片阴影(深浅色都生效)
  - QBird 点睛色由亮青改为更柔和的柔青(#4d8794),与全家桶统一
  - 旧的「素纸 / Monet」主题已退役;升级后自动切到 Lovable(可在 设置 → 主题 改回其它皮肤)
- 设置界面改成 **Obsidian 式分类导航**:左侧按 外观 / 账户 / 模型 / 插件 / 高级 / 关于 分组,找设置更快
- 插件设置面板升级:支持声明式「分区 / 提示 / 外链」等控件,插件 UI **自动跟随主题**(换肤 / 明暗 / 扁平都一致)——为插件统一外观打基础

## 1.3.1 (2026-06-25)

- 新增「应用内更新」:设置 → 关于 可一键检查更新;检测到新版会在主界面顶部提示
  - Windows / Linux:可直接在应用内下载并重启安装,无需手动去下载页
  - macOS:检测到新版后引导前往 Releases 页手动下载(暂未启用 mac 应用内自动安装)
  - 启动时静默检查一次;下载与安装均需你点击确认

## 1.3.0 (2026-06-25)

- 新增「外部 Agent 引擎」:新会话开始处可选用 Claude Code、Codex 等第三方 AI Agent 框架代替内置 Tangu 引擎(经 ACP 接入)——
  - 像用 Tangu 一样在主界面对话,自动适配该引擎的模型选择与 slash 命令
  - 设置 → Agent CLIs:查看本机已检测到的引擎,并为每个引擎设默认模型
  - 引擎选择器仅在对应引擎「已检测到」时出现;一个会话从始至终使用同一种引擎
- 新增「群聊模式」:让多个智能体围绕同一话题轮流发言、投票,并可由主持人总结(本地工作区)
- 新增「官方账号订阅登录」:用你的 Claude / ChatGPT 订阅额度跑 Tangu(OAuth 登录,无需 API Key)
- 记忆改为「本地优先」:记忆与日志默认存在本机(~/.tangu),可在设置里与 Forsion 云端双向同步(默认手动,隐私优先)
- 代码编辑更稳:新增结构化补丁(apply_patch)与本机文件操作安全闸(路径越界需确认)
- 修复:已安装的 Claude Code / Codex 偶尔检测不到;云端会话不再误列本地模型、也不再显示外部引擎(外部引擎仅本地会话可用,且与群聊互斥)

## 1.2.0 (2026-06-21)

- 新增「微信远程」:在 设置 → 微信远程 扫码连接你的微信 bot,从微信里直接和 Tangu Agent 对话——
  - 作为一个独立的工作区(Project)出现在侧栏顶部,可折叠展开它下面的会话
  - 微信里可用 `/new` 新建会话、`/list` 列出、`/switch <序号>` 切换、`/help` 帮助;回复「停止」中止任务、「批准 / 拒绝」处理待批操作;有「正在输入」状态
  - Agent 可把工作区里的文件 / 图片直接发回到你的微信(「把这个文件发我微信」)
  - 主界面用于设置 / 连接,可查看与切换「正在连接的会话」
- 新增「浏览器工具」:Agent 可用本地轻量浏览器实时搜索网页并交互(`browser_search` 等),默认 DuckDuckGo,支持 Bing / Google / 百度
- 自定义 Provider 新增「拉取模型」:填了 Base URL / API Key 后一键拉取该端点可用模型列表,搜索勾选即可
- 侧栏改进:微信连接默认置顶;工作区可拖拽调整排序;新增会话搜索;删除以「归档」为主操作(永久删除收进归档区)
- 聊天区新增「划线引用」:选中文字浮出「引用」,作为引用块拼到下一条消息前(类 ChatGPT)
- 顶栏精简:去掉一排状态胶囊(Host / 微信 / 在线 等),只在掉线时提示
- 更顺手:微信来消息、开关后台智能体等变化现在自动呈现,无需手动刷新页面
- ⚠️ 「后台智能体」Historian(历史员)与 Muse(缪斯)仍为**实验性功能,目前极其不稳定**,可能出现卡住、重复触发、消耗异常等问题;默认关闭,请谨慎在非重要工作区试用,正式使用前建议保持关闭

## 1.1.0 (2026-06-18)

- 新增「上下文用量条」:输入框下方实时显示当前模型上下文占用百分比与本会话 token 消耗,一眼看清还剩多少额度
- 新增「压缩上下文」:对话变长时一键(输入栏按钮或 `/compact`)总结后精简续接;接近上限会自动压缩,避免超长报错、长对话也能稳稳继续
- 新增「智能体」(Normal Agent):在 设置 → 智能体 里创建可复用的对话人格(系统提示词 + 模型 + 设置),用 `/agent` 一键切换;也可让 AI 在对话中自己沉淀出新的智能体
- 新增「后台智能体」(默认关闭,在 设置 → 后台智能体 开启并选模型;记录隔离,不出现在会话列表):
  - Historian(历史员):对话进行中自动总结会话标题,并在需要时维护你的长期记忆与日志
  - Muse(缪斯):后台持续思考「现在能为你做点什么」,产出 TODO 清单;可多选任务、选一个会话一键注入并开始执行
- 侧栏新增「后台智能体」入口:进入 Historian / Muse 工作区,查看它们的活动与 TODO

## 1.0.3 (2026-06-17)

- 修复:本地工作区的文件操作(预览、打开目录、重命名、删除、在文件管理器中显示、拖出)全部失效、点击即报错的问题
- 新增:设置 → 高级「导出会话日志」,将当前会话的全部对话与后端运行日志打包为一个 JSON 文件,便于反馈与排障
- macOS:左上角 Logo 与「Tangu」字样右移,不再被窗口左上角的红绿灯按钮遮挡
- Windows:移除顶部默认菜单栏(文件/编辑/视图/窗口),与 macOS 观感统一
- 新增 `/loop <轮数>` 指令调节单次对话的最大循环轮数(默认上限提升至 90 轮),输入栏与 TUI 均可用
- 循环耗尽(到达轮数上限仍未完成)时给出明确提示,可发送「继续」或用 `/loop` 调高上限
